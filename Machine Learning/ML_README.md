# Machine Learning


***

It is a branch of Artificial Intelligence that enables computers to learn **patterns** from data and make decisions or predictions without being explicitly programmed for every task.
Machine learning is used to solve real-world problems automatically.

Real-Life Examples 

## Machine learning is used in: 

1. YouTube recommendations
1. Netflix movie suggestions
1. Email spam detection
1. Face recognition
1. Chatbots
1. Self-driving cars
1. Stock prediction

## Basic Workflow of Machine Learning

### Step 1: Collect Data

Example:

* Student marks
* House prices
* Customer purchases

### Step 2: Train Model

The algorithm studies patterns in data.

### Step 3: Make Predictions

The trained model predicts new outputs.

Example:

* Predict house price
* Predict exam score

## Types of Machine Learning

There are mainly 4 types of machine learning.

### 1. Supervised Learning

The model learns using labeled data. Labeled data means:

Input + correct output already known

<img width="450" height="100" alt="image" src="https://github.com/user-attachments/assets/cf3e50e9-4ac8-4267-bc3d-0fc654a0857d" />


**The machine learns:** More study hours → better marks

#### 1.1 Goal
 
* Predict outputs for new data.

#### 1.2 Examples

* Email spam detection
* House price prediction
* Student marks prediction

#### 1.3 Common Algorithms

1. Linear Regression
1. Logistic Regression
1. Decision Tree
1. Random Forest


### 2. Unsupervised Learning

The model learns from unlabeled data. No correct answers are provided.

The machine tries to find:

* Patterns
* Groups
* Relationships

#### 2.1 Goal

Find hidden patterns.

#### 2.2 Examples
* Customer segmentation
* Recommendation systems
* Market analysis

#### 2.3 Common Algorithms
1. K-Means Clustering
1. PCA
1. Hierarchical Clustering

### 3. Reinforcement Learning

The model learns by:

* Trial and error
* Rewards and penalties

Similar to how humans learn from experience.

#### 3.1 Example

A robot learns:

* Correct move → reward
* Wrong move → penalty

#### 3.2 Applications
* Self-driving cars
* Robotics
* Game AI

## Advantages of Machine Learning

✅ Automation
✅ Faster predictions
✅ Handles large data

## Limitations of Machine Learning

❌ Requires large data
❌ Can make biased predictions
❌ Training may be expensive
❌ Difficult to understand complex models
