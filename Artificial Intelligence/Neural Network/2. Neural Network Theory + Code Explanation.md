### 3.2 Hidden Layers 
  
### 3.2.1 Hidden Layer 1 

#### Code
```diff
-Dense(8, activation='relu')
```
#### Theory 

Creates 8 neurons.

<img width="200" height="250" alt="image" src="https://github.com/user-attachments/assets/9cb697e7-54d9-4a37-8e6d-de2e4aff4f79" />
<br/>                     

#### Each neuron receives: Hours, Attendance and calculates:

z=w1x1+w2x2+b

Where:

* x1 = Hours
* x2 = Attendance
* w = weights
* b = bias

#### Example:

Hours = 5       , Weight1 = 0.5 ,  
Attendance = 80 , Weight2 = 0.2

z = (5×0.5) + (80×0.2)

* The neuron combines information and learns patterns.

### ReLU Activation 

#### Code

```diff
activation='relu'
```
#### Theory

ReLU removes negative values.

Mathmatical representation: ReLU(x)=max(0,x)

#### Examples:

* ReLU(10) = 10
* ReLU(-5) = 0

#### Why?
* Without activation functions, the network behaves like a simple equation and cannot learn complex patterns.

### 3.2.2 Hidden Layer 2

#### Code

```diff

```

#### Theory

Output from Layer 1 becomes input to Layer 2.

Layer 1 (8 neurons)  ----->  Layer 2 (6 neurons)

This layer learns more abstract features.

#### Example:

Layer 1 may learn:
* Study habit
* Attendance quality

Layer 2 may combine them into: Chance of success

Deep networks learn increasingly complex patterns.

## 6. Compile Model

#### Code

```diff
model.compile(
    optimizer='adam',
    loss='binary_crossentropy',
    metrics=['accuracy']
)
```
#### Theory

##### Optimizer : optimizer='adam' : Adam updates weights automatically.

Think:

Wrong prediction
      ↓
Adjust weights
      ↓
Better prediction

#### Loss Function : loss='binary_crossentropy' : Measures how wrong predictions are.

#### Example:

* Actual = 1
* Prediction = 0.95
* Loss = Small
---
* Actual = 1
* Prediction = 0.10
* Loss = Large

The goal of training is to minimize loss.

## 7. Train Model

#### Code
```diff
- model.fit(
-    X,
-    y,
-    epochs=100,
-    batch_size=1
-)
```
#### Theory

##### Each epoch means: See all training data once

If epochs=100,  the network studies the dataset 100 times.

##### Training cycle:

Forward Pass  -->  Prediction --> Loss Calculation --> Backpropagation --> Weight Update --> Repeat

## 8. Backpropagation (The Heart of Learning)

Suppose:

* Actual = 1
* Predicted = 0.2

Large error.

The network traces backward:

<img width="110" height="160" alt="image" src="https://github.com/user-attachments/assets/9983ad4e-d522-4170-9bd7-dc9fc2dba27e" />


and adjusts weights that contributed to the mistake.

This process is called backpropagation.

Make Predictions

     ↓

What Actually Gets Learned?

The neural network learns weights.

Initially:

Random Weights

Example:

Hours Weight = 0.12
Attendance Weight = -0.43

After training:

Hours Weight = 1.85
Attendance Weight = 0.97

The network discovers which inputs matter and by how much.

Mental Model

Think of a neural network as a giant function:

Inputs
   ↓
Weighted Calculations
   ↓
Activation Functions
   ↓
Hidden Layers Learn Patterns
   ↓
Output Probability

The hidden layers are feature detectors, the weights store learned knowledge, and backpropagation continuously improves those weights based on errors. That's the core idea behind most modern AI systems.
