Point to note based on above data:-

* There are 3 featurs / input columns (Sleep, Coffee, Time).
* There is one output column (Productivity Score)

Every Neural network has 3 layers.

1. Input Layer
2. Hidden Layer
3. Output layer

# 1. Input Layer

Lets make Input layer based on number of input features.  
### Rule : Input layer will have same number of nodes as the input features.
