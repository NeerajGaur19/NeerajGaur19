# 2. Hidden Layer

Let's make Hidden layer next.

We can take as many as nodes of our choice in a layer. There is no restriction on nodes. But the optimized number of nodes depend on the complexity of data. 
More the number of nodes, more will be processing time.

Also, we can take as many number of hidden layers in neural network. The more hidden layers will take the more deeper neural network becomes (Deep learning).
More the number of layers, more will be processing time and high chances of overfitting (learn the data instead of pattern).

# Neural Network - Building

1. Hidden layer is divided into Summation and Activation operation.
2. Connectors between Input and Hidden layer node is weights (Lets say w1, w2, w3...).
3. Bias are applied on Hidden layer and output layer nodes. Bias helps position the boundary. Almost every neuron in a hidden layer has its own bias term.
