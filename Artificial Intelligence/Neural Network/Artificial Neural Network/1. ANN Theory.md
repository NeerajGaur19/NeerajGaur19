# Artificial Neural Network (ANN)

Think of an ANN as a machine that tries to learn patterns from data the same way a human brain learns from experience.

## Real-Life Example

Suppose you want to predict:
<br> Will a customer churn?

Input:

* Tenure = 2 months
* Monthly Charges = ₹5000
* Contract = Month-to-Month

Output:

Will Churn = Yes

An ANN learns this relationship automatically.

## Inspiration from Human Brain

### Human brain:

Eyes --> Neurons --> Brain Processing --> Decision

## 1. ANN:

Input Features --> Artificial Neurons --> Hidden Layers --> Prediction 

## 1.1 ANN Structure

A simple ANN looks like:

Input Layer --> Hidden Layer 1 --> Hidden Layer 2 --> Output Layer 

Example:

    Tenure
    MonthlyCharges
    Contract
    Gender

      ↓

    Hidden Layer
      
      ↓

    Output

### 1.1.1 Input Layer

These are your features.

Example:

    Tenure
    MonthlyCharges
    TotalCharges
    Contract

Suppose: X.shape returns: (5000, 10)

Meaning: 5000 rows,  10 features

Then: Input Layer = 10 neurons

---

### 1.1.2 Hidden Layer

This is where learning happens.

Example:
  
  Dense(64) means: 64 neurons
  
  Visual:

    10 Inputs
         ↓
    64 Neurons

Each neuron learns a different pattern.

### 1.1.3 What Does a Neuron Do?

Suppose:

* Tenure = 24
* MonthlyCharges = 80

Neuron calculates:

Output = (W1 × Tenure) + (W2 × MonthlyCharges) + Bias

Mathematically:

Example: (0.5 × 24) + (0.8 × 80) + 5 = 81

### 1.1.4 Activation Function

After calculating: 81

ANN asks: "Should I activate this neuron?"

Using: activation='relu'

ReLU means: f(x)=max(0,x)

Examples:

* ReLU(81) = 81
* ReLU(-5) = 0

This introduces non-linearity. 

Without activation: ANN = Linear Regression,  which is very limited.

### 1.1.5 Why Multiple Hidden Layers?

Suppose predicting house price.

Layer 1 learns:

* Area
* Bedrooms
* Bathrooms

Layer 2 learns:

* House Size
* Luxury Level

Layer 3 learns:

* Estimated House Value

Thus:

    Simple Features
          ↓
    Complex Features
          ↓
    Prediction

---

### 1.1.2 Output Layer

### 1.1.2.1 Classification

Churn:

Dense(1, activation='sigmoid')

Output:

0.00 → No Churn
1.00 → Churn

Example:

0.83

means:

83% chance of churn

### 1.1.2.2 Regression

House Price:

Dense(1)

Output:

₹85,00,000

or

250000

continuous value.

### 1.1.3 Forward Propagation

Data moves:

    Input
     ↓
    Hidden Layer
     ↓
    Output

Example:

    Customer Data
     ↓
    ANN
     ↓
    0.82

Prediction: Will Churn


### 1.1.4 Loss Function

ANN checks:  Actual Value vs Predicted Value

Example:

Actual = 1
<br>
Predicted = 0.82

Error exists.

Loss measures this error.

#### Classification
loss='binary_crossentropy'

#### Regression
loss='mse'

### 1.1.5 Backpropagation

This is the most important step.

ANN asks:

Why was I wrong?

Then updates weights.

    Wrong Prediction
          ↓
    Calculate Error
          ↓
    Update Weights
          ↓
    Improve Prediction

### 1.1.5 Optimizer

Optimizer decides:

How should weights change?

Most common:  optimizer='adam'

Adam is like a smart GPS.

Instead of randomly searching:

    Loss = 100
     ↓
    80
     ↓
    50
     ↓
    20
     ↓
    5

Adam quickly moves toward the minimum loss.

### 1.1.6 Epoch

One complete pass through training data.

Example:

epochs=50

#### Means:
    
    Read dataset
    Learn
    
    Read dataset again
    Learn
    
    Repeat 50 times

### 1.1.7 Batch Size

Suppose:

5000 rows

Batch size:

batch_size=32

ANN learns:
    
    32 rows
    Update Weights
    
    Next 32 rows
    Update Weights

...

instead of processing all rows together.

### 1.1.8 Overfitting

ANN memorizes training data.

Example:

Training Accuracy = 99%

Testing Accuracy = 75%

Bad.

### 1.1.9 Dropout

Solution:

Dropout(0.2)

Randomly disables neurons.

    Neuron 1 → Off
    Neuron 5 → Off
    Neuron 9 → Off

during training.

This prevents memorization.

### 1.1.10 Early Stopping

Suppose:
    
    Epoch 10 → Good
    
    Epoch 20 → Better
    
    Epoch 30 → Worse

Stop automatically: EarlyStopping(restore_best_weights=True)
