# Artificial Neural Network (ANN)

Think of an ANN as a machine that tries to learn patterns from data the same way a human brain learns from experience.

## Real-Life Example

### Suppose you want to predict:

Will a customer churn?

### Input:

Tenure = 2 months
Monthly Charges = ₹5000
Contract = Month-to-Month

### Output:

Will Churn = Yes

An ANN learns this relationship automatically.

## Inspiration from Human Brain

### Human brain:

Eyes --> Neurons --> Brain Processing --> Decision

## ANN:

Input Features --> Artificial Neurons --> Hidden Layers --> Prediction 

## ANN Structure

A simple ANN looks like:

Input Layer --> Hidden Layer 1 --> Hidden Layer 2 --> Output Layer 

Example:

    Tenure
    MonthlyCharges
    Contract
    Gender

      ↓

    Hidden Layer
      
      ↓

    Output


### Input Layer

These are your features.

Example:

    Tenure
    MonthlyCharges
    TotalCharges
    Contract

Suppose: X.shape returns: (5000, 10)

Meaning: 5000 rows,  10 features

Then: Input Layer = 10 neurons

### Hidden Layer

This is where learning happens.

Example:
  
  Dense(64) means: 64 neurons
  
  Visual:

    10 Inputs
         ↓
    64 Neurons

Each neuron learns a different pattern.

### What Does a Neuron Do?

Suppose:

* Tenure = 24
* MonthlyCharges = 80

Neuron calculates:

Output = (W1 × Tenure) + (W2 × MonthlyCharges) + Bias

Mathematically:

Example: (0.5 × 24) + (0.8 × 80) + 5 = 81

### Activation Function

After calculating: 81

ANN asks: "Should I activate this neuron?"

Using: activation='relu'

ReLU means: f(x)=max(0,x)

Examples:

* ReLU(81) = 81
* ReLU(-5) = 0

This introduces non-linearity. 

Without activation: ANN = Linear Regression,  which is very limited.

### Why Multiple Hidden Layers?

Suppose predicting house price.

Layer 1 learns:

* Area
* Bedrooms
* Bathrooms

Layer 2 learns:

* House Size
* Luxury Level

Layer 3 learns:

* Estimated House Value

Thus:

    Simple Features
          ↓
    Complex Features
          ↓
    Prediction
